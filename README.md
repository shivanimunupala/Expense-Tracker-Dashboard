# 💰 Expense Tracker Dashboard

A modern Expense Tracker Dashboard built using HTML, CSS, and JavaScript.

## Features

- 🌙 Dark Mode UI
- 💳 Dashboard Cards
- ➕ Add Transactions
- 📊 Expense Analysis Chart
- 📅 Transaction History
- 🗑 Delete Transactions
- 💾 Local Storage
- 📱 Responsive Design

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js

- 📄 Export to PDF
